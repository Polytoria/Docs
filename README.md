## Installation Instructions

1. Install the dependencies with [pip](https://pypi.org/project/pip/) by running `pip install -r .\requirements.txt`
2. Run `mkdocs serve`
3. Input `http://127.0.0.1:8000/` into browser to view locally to edit.
